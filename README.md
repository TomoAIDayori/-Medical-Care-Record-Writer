# Medical & Care Record Writer

「介護・医療記録の作成をもっとスムーズに。文例に悩む時間をゼロに。」

このツールは、介護・医療現場で日々作成される訪問記録や支援内容の記録を、簡単に自動生成するGPTアシスタントです。

## Features
- 記録フォーマットに沿った文章自動生成
- 敬体／常体の切り替え
- 誤字脱字のない自然な日本語
- 個人情報取り扱いに配慮

業務効率化や新人教育のサポートを目的に開発しました。
